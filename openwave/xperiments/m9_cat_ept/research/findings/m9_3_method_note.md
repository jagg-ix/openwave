# M9.3 method note: fixed coarse-graining clocks

## Result

All eleven preregistered checks pass for both the initial and final M9.2
snapshots. The result is a data-processing statement along **channel depth**, not
a theorem that physical time is irreversible.

## Equations

```text
p_i = dx |psi_i|^2 / sum_j dx |psi_j|^2
(Kp)_i = 1/4 p_{i-1} + 1/2 p_i + 1/4 p_{i+1}
D_rem(n) = D_KL(p_n || u)
G_acc(n) = D_KL(p_0 || u) - D_KL(p_n || u)
I_n = D_KL(p_n(x) K(y|x) || p_n(x) p_{n+1}(y))
```

The exact ledger is

```text
D_rem(0) = D_rem(n) + G_acc(n).
```

## Interpretation boundary

The accumulated clock is generated by an explicit Markov coarse-graining step.
No dissipative term is inserted into the M9.2 Schrödinger equation. Therefore
M9.3 supplies a controlled observation/forgetting clock, not yet a dynamical
CAT/EPT imaginary-action back-reaction.
