# AI Analysis: reviews produced for external authors

This folder is the public log of **AI-generated analyses** produced by the OpenWave project as a service to external authors who request them. Each file is written entirely by an AI model at the author's request, relayed by the OpenWave maintainer; the maintainer commissions, places, and publishes the file but does not author its technical content.

## Standing rules for every file in this folder

| Rule | Detail |
| --- | --- |
| Naming | `YYYY-MM-DD_HHMM_name.md` (date and time the analysis was produced) |
| Disclosure | Every file opens with an Intelligence Source Disclosure section: model identity, who commissioned it and why, not-peer-review status, as-is/no-warranty, and the [`AI_HYGIENE.md`](../../../../../AI_HYGIENE.md) framing (AI output is hypothesis until independently verified) |
| Claim tagging | Findings are tagged **[in-document]** (verifiable from the reviewed document alone) or **[verify]** (requires independent confirmation) |
| No hosting of reviewed material | Reviewed manuscripts are never reproduced or hosted here; only brief excerpts are quoted for scholarly criticism. Pre-publication drafts get a DOI link added once the author's record is live |
| Corrections | Via repository issues; author responses are linked from the file they answer |
